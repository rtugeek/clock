### 运行项目

#### 1.下载并运行桌面组件客户端

1. https://www.microsoft.com/store/productId/9NPR50GQ7T53
2. https://widgetjs.cn

#### 2.克隆代码

```shell
#克隆代码
git clone https://github.com/widget-js/widget.git

```

#### 3.到项目目录下载依赖

```shell
pnpm install
```

#### 4.运行

```shell
pnpm dev
```

### 创建自己的组件

```shell
// 安装组件脚手架
pnpm install -g @widget-js/cli
// 在项目根目录运行 创建组件命令
widget create
```
